Суулгах сангууд:

1.**streamlit** - Вэб дээр интерактив аппликейшн (dashboard) хурдан бүтээхэд ашиглагддаг. Текст, хүснэгт, график, зураг гэх мэт контентыг динамикаар харуулах боломжтой. **_(pip install streamlit)_**

2.**pandas** - Өгөгдлийг хүснэгт (DataFrame) хэлбэрээр хадгалах, боловсруулах, тооцоолох.  **_(pip install pandas)_**

3.**numpy** - Тоон өгөгдлийн тооцоолол, массив болон матрицийн үйлдлүүд, статистик функц. _**(pip install numpy)**_

4.**re** - Regular expression ашиглан текстээс өгөгдөл хайх, орлуулах, шалгах. _**(Python стандарт сан, нэмэлт суулгалт хэрэггүй.)**_

5.**sklearn** - Машин сургалтын загвар боловсруулах, түүвэрлэх, үнэлэх. _**(pip install scikit-learn)**_

6.**matplotlib** - 2D болон 3D график зурж дүрслэх. _**(pip install matplotlib)**_
                    - cm – colormap буюу өнгөний схем сонгоход ашиглагддаг
                    - mpl_toolkits.mplot3d – 3D график хийхэд хэрэгтэй.

7.**seaborn** - Статистик график, визуализацийг энгийн, гоё харагдуулах. Matplotlib дээр суурилдаг. _**(pip install seaborn)**_

8.**plotly** - Интерктив график, диаграмм үүсгэх. Streamlit-д хялбар оруулах боломжтой. _**(pip install plotly)**_
